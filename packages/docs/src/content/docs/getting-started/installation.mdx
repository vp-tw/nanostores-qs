## Install the package

<Tabs>
  <TabItem label="npm">```bash npm install @vp-tw/nanostores-qs nanostores ```</TabItem>
  <TabItem label="pnpm">```bash pnpm add @vp-tw/nanostores-qs nanostores ```</TabItem>
  <TabItem label="yarn">```bash yarn add @vp-tw/nanostores-qs nanostores ```</TabItem>
</Tabs>

## Framework bindings

Add the nanostores binding for your framework:

<Tabs>
  <TabItem label="React">```bash npm install @nanostores/react ```</TabItem>
  <TabItem label="Vue">```bash npm install @nanostores/vue ```</TabItem>
  <TabItem label="Svelte">```bash npm install @nanostores/svelte ```</TabItem>
  <TabItem label="Solid">```bash npm install @nanostores/solid ```</TabItem>
</Tabs>

## Entry points

The library exports three entry points:

| Import path                              | Purpose                           |
| ---------------------------------------- | --------------------------------- |
| `@vp-tw/nanostores-qs`                   | Core: `createQsUtils` factory     |
| `@vp-tw/nanostores-qs/presets`           | Built-in presets for common types |
| `@vp-tw/nanostores-qs/defineSearchParam` | Create custom reusable presets    |
